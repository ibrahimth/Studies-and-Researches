O que é

Uma regressão linear é um método que faz "previsões/estimativas" baseado em um conjunto de dados e que utiliza como base uma função de primeiro grau do seguinte formato:

y = ax + b

A técnica é simples e permite realizar estimativas com facilidade porém a precisão cai caso o a variância do conjunto de dados utilizado como base para "previsão/estimativa" seja muito alta. Isso acontece devido ao fato de que uma função de primeiro grau consegue gerar apenas linhas e caso a variância dos dados seja muito alta, uma linha não consegue ser sufienciente para abrangir todos os dados
